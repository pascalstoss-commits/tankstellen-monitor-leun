# Tankstellen-Monitor Leun – Komplettpaket

Dieses Paket enthält die bereinigten Dateien für:
- robusteres Frontend (`index.html`)
- resilienteres Datenskript (`fetch_data.py`)
- stabileren GitHub-Workflow (`.github/workflows/update-fuel-data.yml`)

## Austausch auf GitHub
Ersetze im Repository diese Dateien komplett:
- `index.html`
- `fetch_data.py`
- `.github/workflows/update-fuel-data.yml`
- `requirements.txt`

Optional:
- `DEPLOY_INSTRUCTIONS.md`

## Danach
1. Committen
2. Pushen
3. In GitHub unter **Actions** den Workflow einmal manuell starten
4. In GitHub unter **Settings → Pages** als Source **GitHub Actions** auswählen
